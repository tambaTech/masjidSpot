# MasjidMapView User Guide

## Quick Start

### Finding Mosques Near You

1. **Open the Map** - The map automatically centers on your location
2. **See Nearby Mosques** - Pins show all mosques within your filter range
3. **Quick Navigate** - Tap the blue "Navigate to Nearest" button for instant directions

## Features

### 🔍 Search & Filter

#### Search Bar
- **Type to Search**: Enter mosque name or location
- **Clear Search**: Tap the ✕ button to clear
- **Live Results**: See results update as you type

#### Filter Menu (⚙️ icon)
Access advanced options:

**Map Styles**
- Standard - Default map view
- Satellite - Aerial imagery
- Hybrid - Satellite with labels

**Sort Options**
- Distance - Nearest first (default)
- Name - Alphabetical order
- Recently Added - Newest mosques first

**Distance Filter**
- 5 km - Very close mosques only
- 10 km - Nearby mosques
- 25 km - Local area
- 50 km - Wide area (default)
- 100 km - Extended region

### 🗺️ Map Controls

#### Bottom Right Controls
- **Location Button** 📍
  - Blue = Tracking your location
  - White = Map moved, tap to recenter
  
- **Directions Button** 🚗
  - Green = Mosques available
  - Opens directions options sheet

#### Bottom Left Control
- **Look Around Button** 👁️
  - View street-level imagery
  - See mosque surroundings

### 📍 Mosque Pins

#### Tap a Pin
- View mosque details
- See photo and information
- Get directions
- View contact info

#### Pin Information
- Mosque name
- Distance from you
- Custom mosque icon/photo

### 🚀 Quick Actions

#### Navigate to Nearest
The blue card shows:
- Nearest mosque name
- Distance away
- Tap for instant directions in Maps app

#### Menu Actions
- **Fit All Mosques** - Zoom to show all filtered mosques
- **Look Around** - Street view of current location
- **Refresh Data** - Reload mosque information

### 💬 Notifications

Toast messages appear at top:
- ✅ **Green** - Success (e.g., "Data refreshed")
- ℹ️ **Blue** - Information (e.g., "Centered on your location")
- ❌ **Red** - Error (e.g., "Location not available")

### 📊 Result Counter

When filtering is active, see:
- "Showing X of Y mosques"
- X = Filtered results
- Y = Total mosques

### 🎯 Empty States

**No Results?**
- Check your search term
- Increase distance filter
- Tap "Clear Filters" button

**No Mosques at All?**
- Enable location services
- Check internet connection
- Pull to refresh

## Tips & Tricks

### For Best Results

1. **Enable Location Services**
   - Settings → Privacy → Location Services
   - Allow "MosquePin" to use your location

2. **Adjust Distance Filter**
   - In rural areas, increase to 100 km
   - In cities, decrease to 5-10 km

3. **Use 3D View**
   - Toggle in menu for better spatial awareness
   - See building heights and terrain

4. **Save Time**
   - Use "Navigate to Nearest" for quick directions
   - Bookmark frequently visited mosques

### Gestures

- **Pinch** - Zoom in/out
- **Drag** - Pan around map
- **Two-finger rotate** - Rotate map
- **Two-finger swipe up/down** - Adjust 3D angle

### Battery Saving

- Use Standard map instead of Satellite
- Disable 3D view when not needed
- Turn off location tracking when browsing distant areas

## Troubleshooting

### Map Not Loading
1. Check internet connection
2. Try refreshing data (menu → Refresh Data)
3. Restart the app

### Location Not Found
1. Enable Location Services
2. Grant permission to app
3. Check if GPS is available
4. Try outdoors for better signal

### No Mosques Showing
1. Increase distance filter
2. Clear search text
3. Check if you're in a remote area
4. Refresh data

### Directions Not Working
1. Ensure Apple Maps is installed
2. Check location permissions
3. Verify mosque has valid address

## Privacy

- Only used for finding nearby mosques
- Not stored or shared
- Processed on-device
- Can be disabled anytime

### Search History
- Not saved
- Private to current session
- No tracking or analytics
